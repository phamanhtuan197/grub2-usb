# Install

## Step 1:
Download [grub2-usb](https://github.com/torn4dom4n/grub2-usb/releases).

## Step 2:
Using [`UltraISO`](https://www.ezbsystems.com/ultraiso/) or [`refus`](https://rufus.akeo.ie/) burn out the USB or CD with `mbr`.
